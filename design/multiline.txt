
expr 이 두줄 이상 인 경우 일반적인 tab indent 를 기반으로 트리를 생성한다. 

    예:
        a
            b
            c
                d
    
    => a( b, c.d )



단 개별 라인도 하나의 구조로 파싱을 수행한다. 

    예: 
        a 
            b.c 
            d.e 

        => a(b.c, d.e) 

    예: 
        a.b
            c
         
        => a.b.c 

상위 노드의 다음 하위 노드는 상위 노드의 마지막 단일 노드 ( 형제가 없는 ) 에 귀속된다. 


    예: 
        a.b(c,d)
            e 
        -> a.b(c,d,e)


형제가 없는 노드라도 () 로 묶여있으면 다음 하위 노드가 귀속되지 안는다. 

    예: 
        a.b(c)    
            d
        => a.b(c,d)
